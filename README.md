## 干货数据源
应用中的所有干货数据均来自[干货集中营](http://gank.io/)。

## App设计
色调：远州鼠、落栗、苏芳、石竹、枯草、柳煤竹茶、锖青磁、鸠羽紫八种淡雅的颜色，分别对应不同的干货数据。
布局：简约风格，极简主义，去繁就简。

## UI设计
1. 黑白灰紫蓝
2. 圆角
